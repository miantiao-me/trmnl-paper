# 官方示例索引

以下示例从 [TRMNL Framework v3 Examples](https://trmnl.com/framework/examples?version=v3) 提取，每个文件包含 Full View 的完整 HTML 源码。

每个示例均展示多种 Framework 特性的组合用法，可作为实际开发的参考起点。

| 文件 | 类型 | 演示特性 |
|---|---|---|
| [weather.md](weather.md) | 天气看板 | grid 多列、响应式图片、data-fit-value、item/value 嵌套 |
| [stock_price.md](stock_price.md) | 股票报价 | grid + col--span、value--tnums、data-fit-value、divider 分隔 |
| [todo_list.md](todo_list.md) | 待办列表 | columns + overflow、group-header、index 编号、label--gray |
| [shopify.md](shopify.md) | 电商看板 | grid 混合布局、Highcharts 图表集成、data-value-type |
| [lunar_calendar.md](lunar_calendar.md) | 月相日历 | flex 自定义布局、grid--row、图片序列、label--inverted |
| [days_left_year.md](days_left_year.md) | 年度进度 | grid + JS 动态渲染、data-value-fit、自定义 CSS grid |
| [wiki_random_article.md](wiki_random_article.md) | 富文本阅读 | richtext + content-limiter、pixel-perfect、columns |
| [github_commit_graph.md](github_commit_graph.md) | GitHub 提交图 | grid 密度分布、热力图、自定义颜色块 |
| [reddit.md](reddit.md) | Reddit 热门帖子 | columns + overflow、group-header 分组、index 编号、label--underline 元数据 |
